▼ array
1.
```cpp
int a[5][5] = {{1,2}, {3}};
```
이렇게 되면 `a[0]` 에서 1, 2값 외의 자리에는 0이 들어감. 
- a[0] >> { 1, 2, 0, 0, 0 }
- a[1] >> { 3, 0, 0, 0, 0 }

💡 타입이 `char`일 경우에는, 빈자리에 `null` 값이 들어감.

2.
```cpp
char shr[3][5] = {"삼성", "현대", "대우"};
```

문자열을 받을 경우 shr[0] >> 주소값을 가지고 있음.
반면, shr[0][1] >> ‘A’데이터를 가지고 있음. 그래서 값을 입력받을 때는 번지연산자(&)를 사용해야함
